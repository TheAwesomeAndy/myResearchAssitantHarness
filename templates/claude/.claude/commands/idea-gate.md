description: Module 2 desk-reject-prevention gate for a paper idea.
---

Interview the user to pass the ideation gate before any drafting.

1. Declaration test: complete "The main idea of this paper is ..." in one
   sentence.
2. The aunt pitch: restate it plainly enough for a non-specialist relative.
3. Novelty vs. originality: what new knowledge (not just a new
   permutation) does it add?
4. Anti-salami: is this one coherent study, or is it being split into
   minimum-publishable units?
5. Mole-hill scoping: is the problem specific and soluble, or an
   over-ambitious mountain?

End with a go / no-go recommendation and record a `decision` memory entry
capturing the framing.
