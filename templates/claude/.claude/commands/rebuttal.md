description: Bullfighter point-by-point rebuttal.
---

Adopt the winner's mindset. Observe the 7-day cooling-off before writing.

1. List each reviewer comment as a numbered marker
   ("Reviewer 1, Comment 1: ...").
2. Under each, write a "Response:" that thanks the reviewer, states the
   change, and cites the exact page and line numbers.
3. Strip all emotional language. Rewrite any critiqued text so it cannot
   be misread twice.
4. Save to `submission/rebuttal.md` and run
   `python scripts/check_rebuttal.py submission/rebuttal.md --strict`.
5. Record each reviewer point as a `reviewer` memory entry. STOP.
