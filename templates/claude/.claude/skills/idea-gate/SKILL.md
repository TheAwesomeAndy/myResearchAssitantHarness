# Idea Gate

Use before any drafting begins, to prevent a desk reject.

## Procedure
1. Declaration test: one sentence, "The main idea of this paper is ...".
2. Aunt pitch: restate for a non-specialist.
3. Novelty vs. originality: new knowledge, not a new permutation.
4. Anti-salami: one coherent study, not split units.
5. Mole-hill scoping: specific and soluble.

## Verification
All five pass and the framing fits one clear sentence.

## Stop point
Give a go / no-go and record a `decision` memory entry.
