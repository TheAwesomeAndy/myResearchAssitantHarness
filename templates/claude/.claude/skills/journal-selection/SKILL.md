# Journal Selection

Use when the user needs a submission target.

## Procedure
1. Parse the reference list; count journals appearing two or more times.
2. For each candidate, verify current aims and scope, typical
   time-to-publication, and open-access options.
3. Screen for predatory signals (unsolicited invitations, poor site,
   opaque fees, fake metrics).
4. Rank by fit with the paper's core idea.

## Verification
Each candidate has scope, turnaround, OA status, and a predatory-screen
verdict recorded.

## Stop point
Present the ranked shortlist and record a `decision` memory entry for the
chosen venue and its blinding policy.
