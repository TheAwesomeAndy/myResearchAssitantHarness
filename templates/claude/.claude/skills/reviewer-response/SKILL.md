# Reviewer Response

Use when handling a decision letter or reviewer comments.

## Procedure
1. Split the letter into numbered comment markers.
2. Map each comment to a concrete manuscript action or a stated,
   defended limitation.
3. Draft a courteous response citing exact page and line numbers.
4. Record each comment as a `reviewer` memory entry (substance only).
5. Run `python scripts/check_rebuttal.py submission/rebuttal.md --strict`.

## Verification
Every comment has a response with a locator; no emotional language; the
rebuttal gate passes.

## Stop point
Present the rebuttal and the list of manuscript edits, then stop.
